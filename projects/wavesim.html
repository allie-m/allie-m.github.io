Spring 2020 was a surprisingly productive time for me. I picked up Metal and, unsatisfied with the state of metal-rs at the time, made my own bindings for fun.

My 8th grade was coming to a close, and I had to do a project for my Science Studio class. The specification was very open-ended (quarantine!), so I decided I would incorporate my surge of interest in GPU programming: make an interactive demo, a 3d wave simulator. It wasn't exactly necessary to use compute shaders or indeed a library as low level as Metal but I felt like it!

The code is available on GitHub. It's very bad, and the demo was cooked up pretty hastily. I haven't tested it since 2020 or on anything newer than macOS High Sierra (10.13), and don't have a computer running macOS to test it on, so I have no idea if it still works.

According to GitHub there are some serious security vulnerabilities in the versions of the dependencies the project is locked to, so... make of that what you will.

Still, it was a lot of fun to make.
